Help & Documentation

📤 Getting Started - Uploading Your Data

To use Claude Explorer, you need to export your conversations from Claude.ai:

  1. Go to claude.ai
  2. Click your profile icon in the bottom-left corner
  3. Select "Settings" from the menu
  4. Navigate to the "Data" or "Privacy" section
  5. Click "Export Data" and download the ZIP file
  6. In the app, click Choose File and upload the ZIP
  7. Your data is automatically saved to ~/.claude-explorer/data/ and persists across restarts

🔍 Search Features

Basic Search: Type keywords in the search box and press Enter or click Search.

Search Tips:

  • Search looks through conversation names and message content
  • Results show matching snippets from messages
  • Use specific keywords for better results
  • Clear search by searching with empty text

⚙️ Advanced Filters

Click "Advanced Filters" to access powerful filtering options:

  • Date Range: Filter conversations by creation date
  • Message Count: Set minimum and maximum message thresholds
  • Sort Options: Sort by date, message count, or name
  • Only with messages: Hide empty conversations

📊 Analytics Dashboard

The Analytics tab provides insights into your conversation patterns:

  • Activity Over Time: See when you're most active
  • Message Distribution: View short, medium, and long conversations
  • Top Keywords: Discover your most discussed topics (click to search)
  • Conversation Trends: Track weekly and monthly activity

📥 Export Options

Export conversations in multiple formats:

  • 📄 Markdown (MD): Human-readable format, great for documentation
  • 📋 JSON: Raw data format for processing or backup
  • 📦 ZIP Bundle: Complete package with all artifacts and metadata
  • Batch Export: Select multiple conversations and export together

How to export:

  • Single: Click format button (MD/JSON/ZIP) on any conversation card
  • Multiple: Check conversations, then click "Export Selected"

🤖 AI Assistant

The AI Assistant helps you analyze and organize your conversations:

  • Search: "Find conversations about React from 2024"
  • Details: "Show me conversation details for [UUID]"
  • Export: "Export conversations about TypeScript as markdown"
  • Bundles: "Create a knowledge base about authentication"
  • Analysis: "What topics do I discuss most?"

Setup Required:

  1. Get an API key from openrouter.ai/keys
  2. In the AI Assistant tab, click Settings
  3. Paste your API key and click Refresh to load models
  4. Select any model from the dropdown (free and paid options available)
  5. Click Save Settings and start chatting

Powered by OpenRouter — supports GPT-4o, Claude, Gemini, Llama, and hundreds more models.

❓ Frequently Asked Questions

How often should I export my data?

Export your data whenever you want to backup or analyze your conversations. Your uploaded data is automatically saved to ~/.claude-explorer/data/ and persists across server restarts — no need to re-upload after every restart.

Can I search for specific dates?

Yes! Use the Advanced Filters to set a date range for your search.

Where is my data stored?

All data is stored locally on your machine at ~/.claude-explorer/data/. Your conversations never leave your computer unless you explicitly export and share them. You can view the exact path in the Upload section of the app.

What's the difference between export formats?

Markdown is human-readable, JSON contains raw data for processing, and ZIP bundles include everything (messages, artifacts, metadata) in a structured archive.

Can I delete or reset my data?

Yes! Use the Reset Data button in the Upload section to clear all uploaded data from disk. This will remove everything from ~/.claude-explorer/data/ and revert to the default empty state. You can then upload a fresh ZIP.

Do I need an API key to use the app?

No — the web interface works without an API key. You only need an OpenRouter API key if you want to use the AI assistant features (natural language queries, smart search). You can set your key in the Settings panel.
